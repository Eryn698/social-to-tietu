【是什么】
  一键将抖音/小红书图文  →  微信公众号贴图草稿

【怎么用】
  1. 双击「启动.bat」
  2. 浏览器自动打开，粘贴链接，选公众号，点创建 ✅
  3. 去公众号后台 → 草稿箱 查看

【手机上用】
  方式1（局域网 - 最稳）：
    手机和电脑连同一个WiFi
    电脑双击「启动.bat」
    手机打开浏览器，输入 http://你的电脑IP:5000
    （电脑IP查看：Win+R → cmd → ipconfig → IPv4）

  方式2（公网 - 任何网络都能用）：
    双击「启动.bat」
    再双击「启动公网穿透.bat」
    把显示的链接发给手机

【添加公众号】
  页面右上角 ⚙️ 管理 → 输入名称/AppID/AppSecret → 自动验证

【依赖】
  - Python 3.9+（没有的话去 https://www.python.org 下载）
  - cpolar（公网穿透需要，在 C:\Program Files\cpolar\）

【文件说明】
  启动.bat          ← 一键启动（双这个！）
  启动公网穿透.bat    ← 手机远程访问
  server.py         ← 后端
  templates/        ← 前端页面
  scripts/          ← 核心解析&上传脚本
  server/data/      ← 公众号配置存储
